# Task Tracking System

## Overview

Task Tracking System is a web-based application developed using Python, MySQL, and Data Analytics. The system helps organizations manage projects by dividing them into multiple tasks and assigning them to employees.

## Technologies Used

* Python
* MySQL
* Data Analytics
* HTML
* CSS
* JavaScript

## Features

* Admin Dashboard
* Employee Dashboard
* Project Creation and Management
* Task Assignment to Employees
* Task Progress Tracking
* Task Status Updates
* Automatic Email Notifications
* Performance Monitoring
* Data Analytics Reports

## User Roles

### Admin

* Create Projects
* Divide Projects into Tasks
* Assign Tasks to Employees
* Monitor Task Progress
* Generate Reports

### Employee

* View Assigned Tasks
* Update Task Status
* Mark Tasks as Completed
* Receive Task Notifications

## Email Notification System

When a task is completed, the system automatically sends email notifications to both the employee and the administrator.

## Database

MySQL is used for storing employee, project, task, and progress data.
